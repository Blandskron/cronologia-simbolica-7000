# cronologia-simbolica-7000
Representación simbólica estructurada del modelo 7000 años (4000+2000+1000) sobre eje masorético. Integra Biblia, historia de la cristiandad y hitos humanos modernos (Nicea, guerras, ONU, era digital, IA) con escenarios “hipotéticos” como lectura comparativa. No es predicción.
